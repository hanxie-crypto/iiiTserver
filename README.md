基于express以及node 6.0版本的 web项目框架,数据库使用mysql


####示例:  


###特性
主要测试node 6.0对es6的支持情况，借助es6的数组对象扩展，实现了可扩展路由routes.index.js。
数据库方面没有使用odm，orm类的库，使用原生的sql语句。

### 下载安装

git clone
